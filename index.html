Apresentando tags de listas não ordenadas

Para iniciar uma lista não ordenada, é utilizada a tag ul.

  • E para cada item da lista
  • É utilizada uma tag li

Apresentando tags de listas ordenadas

Já uma lista ordenada é inicializada com a tag ol.

  1. E assim como a não ordenada
  2. Cada item receberá a sua tag li
  3. E a cada item, um número seguirá para ordenar a lista
