Classes e IDs CSS

Com a utilização de classes e ids é possível especificar qual TAG receberá a formatação desejada.

Classes são identificadas através do ponto (.) no código CSS.

Ids são identificados através do cerquilha (#).

Normalmente os ids são chamados uma única vez dentro do código, pois são mais utilizados como identificadores de conteúdo. Tanto para o CSS quanto para o JavaScript, costumam armazenar informações, por exemplo. Mas ainda assim, pode ser utilizado quantas vezes quiser.
