Tags Div e Span

A tag div é utilizada para dividir os conteúdos de uma página web, criando diversos blocos de conteúdos ontem todos recebem formatações personalizadas.

A tag span é utilizada para criar elementos inline. Assim como a div, ele também divide o elemento, porém, mantém tudo dentro do mesmo espaço.

Área dedicada para o rodapé da página.
