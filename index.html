Introdução ao CSS (Cascading Style Sheet)

O CSS é utilizado para formatar os conteúdos exibidos em páginas web.

Existem três maneiras de se utilizar o CSS. São elas:

  • Inline: em conjunto com as tags dentro do próprio arquivo HTML
  • Internal: escrevendo um bloco de código separado das tags mas ainda sim dentro do próprio arquivo HTML
  • External: criando um arquivo .css separado do HTML e fazendo ambos se conectarem através da tag link
