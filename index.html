Apresentando tag imagens

Para utilizar imagens em uma página web, é utilizada a tag img. Ela é uma tag que não necessita de fechamento.

Essa tag possui alguns atributos importantes. São eles:

  • src = "CAMINHO_DA_IMAGEM": indica o caminho onde é possível encontrar a imagem.
  • width = "VALOR NUMÉRICO EM PIXELS": define a largura da imagem
  • height = "VALOR NUMÉRICO EM PIXELS": define a altura da imagem

Essa imagem está utilizando todos os atributos e está disponibilizada dentro da pasta do projeto.
