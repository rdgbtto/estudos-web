Bordas com CSS

A utilização do atributo border se dá através de três elementos.

  • Espessura: utilizando em pixels
  • Tipo: indica qual estilo de borda será criado
  • Cor: indica a coloração que a borda recebera

Independentemente da ordem que for inserido, todos os elementos receberão o mesmo efeito.

Também é possível separar cada valor do atributo border alterando para seus atributos específicos.

  • border-width
  • border-style
  • border-color
