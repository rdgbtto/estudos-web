Apresentando tags de formatação de texto

A tag b é utilizada para formatar o texto, deixando em negrito. Porém, ficou obsoleta, e agora por padrão é utilizada a tag strong

A tag i é utilizada para formatar o texto, deixando em itálico. Mas, assim como a tag b, a tag i também ficou obsoleta, sendo substituída pela tag em

A tag u é usada para inserir underline no texto

A tag strike (strike)faz um traço no texto.
