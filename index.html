Apresentando tags de tabela

As tabelas são estruturas bastante utilizadas para apresentar conteúdos na web.

Para criar uma tabela, deve-se utilizar a tag table

Dentro dessa tag serão inseridos outras tags. São elas:

  • th: cabeçalho da tabela (table head)
  • tr: linha da tabela (table roll)
  • td: dado da tabela (table data)
